Vex "In The Zone" Simulation

Install on Windows Install on OSX

Windows Install

   If you just want to check out the finished product and see it in action, simply download the folder from HERE. Make sure all contents of the folder are installed or else the program will not start. Once downloaded, simply run the .exe file and you should be ready to go.

   To build, we'll need to install Cinder C++. This is an open source C++ library that I'm using for graphics and other openGL functions. Specifically with this build you'll want to download Cinder 8.0 or above, but not the developmental 9.0.

OSX install

   If you just want to check out the finished product and see it in action, like the Windows installation simply download the folder from HERE instead. Make sure all contents of the folder are installed or else the program will not start. Once downloaded, simply run the application file and you should be ready to go.

   To build, we'll need to install Cinder C++. This is an open source C++ library that I'm using for graphics and other openGL functions. Specifically with this build you'll want to download Cinder 8.0 or above, but not the developmental 9.0.
